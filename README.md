Two command-line programs made in C++ and working in both Windows and Linux, that add or remove the BOM mark in UTF8 files. 

The program names are:
  * bom_add
  * bom_remove

Example of usage:

    $ bom_add file.txt
    $ bom_remove *.txt *.cpp
